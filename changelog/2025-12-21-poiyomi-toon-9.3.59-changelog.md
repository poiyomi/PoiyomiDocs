This update includes more bug fixes. **In case you missed it, Poiyomi Toon has been graduated to version 9.3. See the previous update patch notes for more info.**

The Toon version is now available to download in our Discord server. Please test it out! As soon as we can confirm this version is bug-free, this will be the version released on the VCC, GitHub, and on BOOTH.

Poiyomi Pro Users: Download the latest Pro version from [pro.poiyomi.com](https://pro.poiyomi.com) by logging in with your Patreon account.

### Changes
- Fixed UV Tile Discard UI bugs.
  - Asterisk was not showing up when changes are made.
  - Reset wasn't resetting all of the Buttons.
- Fixed extra Directional Lights with Shadows enabled causing compiler issues in Lil Fur.
- Fixed Constellation Lines were disappearing in Worlds with Audio Link.
- Fixed Constellation AL Twinkle not working as intended.
- All 3 Post Processing modules now function with Outlines.
- Potentially fixed an issue with the World shader going pink when Emissions aren't active.
  - I'm not 100% sure if this fix will work but it seemed to during my basic testing.
