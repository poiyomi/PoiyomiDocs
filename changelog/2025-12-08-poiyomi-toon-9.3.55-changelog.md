This update includes fixes to some bugs that were discovered in the previous update. **In case you missed it, Poiyomi Toon has been graduated to version 9.3. See the previous update patch notes for more info.**

The Toon version is now available to download in our Discord server. Please test it out! As soon as we can confirm this version is bug-free, this will be the version released on the VCC, GitHub, and on BOOTH.

Poiyomi Pro Users: Download the latest Pro version from [pro.poiyomi.com](https://pro.poiyomi.com) by logging in with your Patreon account.

### Changes
- Fixed Override Two Pass Color always being active.
- Added better defaults for Distortion UVs.
- Fixed up some default settings with the Lil Fur shader.
- Fixed AO and UV Tile Discard for Lil Fur.
- Fixed Decal 3 having an extra Z field for position.
- Fixed Panosphere UV being stereo incorrect when tiled.
- Fixed Vertex Heightmap being removed when height was 0, but Audio Link actually made it not 0.
- Proximity Color works with Lil Fur now.
- Fixed Lil Fur incompatibility with d4rk Avatar Optimizer.
- Proximity Color now properly hides Specular.
- Audio Link Override now properly overrides Audio Link in all cases.
